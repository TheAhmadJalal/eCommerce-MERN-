# eCommerce-MERN-
# Following commands you will need to run in the front-end and back-end directories


##backend:

        npm init -y
        
        npm i express socket.io bcrypt cloudinary  dataenv stripe validator cors
        
        npm i nodemon
        
        npm run dev

    npm i react-thunk
    
    npm i react-redux
    
    npm i axios
    
    npm i react-alice-carousel
    
    npm i @stripe
